# Regular Expressions

- Number -> `[\+\-]?[0-9]+(\.[0-9]+)?`
- String -> ` "[~"]*"`
- Reserved_Keywords -> `int | float | string | read | write | repeat | until | if | elseif | else | then | return | endl | end | main`
- Arithmetic_Operators -> `\+|-|\*|\/`
- Assignment_Operator -> `:=`
- Delimiters -> `{|}|;|.|,`
- Condition_Operators -> `< | > | = | <>`
- Boolean_Operators -> `&& | \|\|`
- Identifiers -> `[A-Za-z][A-Za-z0-9]*`
